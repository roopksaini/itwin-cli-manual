# itwin-cli itwin delete

Delete the specified iTwin.

## Options

- **`--id`**  
  The ID of the iTwin to be deleted.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itwin-cli itwin delete --id "12345abcde"
itwin-cli itwin delete --id "67890fghij"
```
